# 🚀 快速安装指南

## 5分钟快速开始

### 步骤1：准备环境

确保您的Agent后端正在运行：

```bash
cd D:\douyin-agent
python run_backend.py
```

### 步骤2：安装扩展

1. **打开Chrome扩展管理**
   - 在地址栏输入：`chrome://extensions/`

2. **启用开发者模式**
   - 点击右上角的"开发者模式"开关

3. **加载扩展**
   - 点击"加载已解压的扩展程序"
   - 选择 `D:\douyin-agent\browser-extension` 文件夹

4. **确认安装**
   - 看到扩展图标出现在工具栏中 ✅

### 步骤3：测试功能

1. **打开抖音创作中心**
   - 访问：https://creator.douyin.com/creator-micro/content/manage

2. **检查扩展状态**
   - 点击扩展图标
   - 确认显示"抖音创作中心"和"已连接"

3. **提取数据**
   - 点击"提取数据"按钮
   - 查看数据预览

4. **导入数据**
   - 点击"确认导入"
   - 等待成功提示

## 🎯 常用操作

### 一键导入
1. 在抖音创作中心页面
2. 点击页面右上角的"📥 导入到Agent"按钮
3. 确认数据后导入

### 右键菜单
1. 在抖音创作中心页面右键
2. 选择"📥 导入当前页面数据"
3. 自动完成导入

### 批量导入
1. 打开包含多个视频的页面
2. 使用扩展提取所有数据
3. 批量导入到系统

## ⚙️ 基本设置

### 修改后端地址
1. 打开扩展popup
2. 在"设置"中修改"后端地址"
3. 保存后自动测试连接

### 启用自动导入
1. 打开扩展设置
2. 将"自动导入"设置为"开启"
3. 提取数据后自动导入

## 🔍 故障排除

### 扩展不工作？
1. 刷新抖音页面
2. 检查扩展是否启用
3. 查看浏览器控制台错误

### 连接失败？
1. 确认后端正在运行
2. 检查后端地址设置
3. 测试后端API：`http://127.0.0.1:9000/api/health`

### 数据提取失败？
1. 确认在正确的页面
2. 等待页面完全加载
3. 尝试不同的页面（内容管理、视频详情）
