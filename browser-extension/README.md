# 抖音创作中心数据导入助手

## 📖 简介

抖音创作中心数据导入助手是一个浏览器扩展，帮助您从抖音创作中心页面自动提取发布数据并导入到Agent系统，形成完整的数据闭环。

## ✨ 主要功能

- **自动数据提取**：智能识别抖音创作中心页面的发布数据
- **一键导入**：点击按钮即可将数据导入到Agent系统
- **数据预览**：导入前可预览和确认数据
- **批量处理**：支持批量提取多条发布记录
- **实时状态**：显示页面状态和后端连接状态
- **右键菜单**：支持右键菜单快速导入

## 🚀 安装方法

### 方法1：开发者模式安装（推荐）

1. **下载扩展文件**
   - 下载整个 `browser-extension` 文件夹

2. **打开Chrome扩展管理页面**
   - 在Chrome浏览器中输入：`chrome://extensions/`
   - 或者：菜单 → 更多工具 → 扩展程序

3. **启用开发者模式**
   - 点击右上角的"开发者模式"开关

4. **加载扩展**
   - 点击"加载已解压的扩展程序"
   - 选择 `browser-extension` 文件夹

5. **确认安装**
   - 看到扩展图标出现在浏览器工具栏中
   - 点击图标确认扩展正常工作

### 方法2：打包安装

1. **打包扩展**
   - 在扩展管理页面点击"打包扩展程序"
   - 选择 `browser-extension` 文件夹
   - 生成 `.crx` 文件

2. **安装扩展**
   - 将 `.crx` 文件拖拽到Chrome扩展管理页面
   - 确认安装

## 📋 使用方法

### 基本使用流程

1. **启动Agent后端**
   ```bash
   cd D:\douyin-agent
   python run_backend.py
   ```

2. **打开抖音创作中心**
   - 访问：https://creator.douyin.com/creator-micro/content/manage
   - 登录您的抖音账号

3. **打开扩展**
   - 点击浏览器工具栏中的扩展图标
   - 检查页面状态和后端连接状态

4. **提取数据**
   - 确认状态正常后，点击"提取数据"按钮
   - 查看数据预览

5. **导入数据**
   - 确认数据无误后，点击"确认导入"
   - 等待导入完成

### 高级功能

#### 右键菜单导入

1. 在抖音创作中心页面右键点击
2. 选择"📥 导入当前页面数据"
3. 自动提取并导入数据

#### 自动导入设置

1. 打开扩展设置
2. 启用"自动导入"选项
3. 提取数据后自动导入，无需确认

#### 自定义后端地址

1. 打开扩展设置
2. 修改"后端地址"为您的Agent服务地址
3. 保存设置

## 🔧 配置说明

### 扩展设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 后端地址 | Agent系统的API地址 | http://127.0.0.1:9000 |
| 自动导入 | 提取数据后是否自动导入 | 关闭 |
| 通知启用 | 是否显示导入成功通知 | 开启 |

### 后端要求

确保您的Agent后端正在运行并且可以访问：

```bash
# 检查后端状态
curl http://127.0.0.1:9000/api/health

# 检查同步接口
curl -X POST http://127.0.0.1:9000/api/douyin/sync \
  -H "Content-Type: application/json" \
  -d '{"records": []}'
```

## 🐛 故障排除

### 扩展无法注入到页面

**问题**：在抖音创作中心页面看不到导入按钮

**解决方案**：
1. 刷新页面
2. 检查扩展是否已启用
3. 确认当前URL是否为 `creator.douyin.com` 或 `creator.douyin.cn`
4. 查看浏览器控制台是否有错误

### 后端连接失败

**问题**：显示"未连接"状态

**解决方案**：
1. 确认Agent后端正在运行
2. 检查后端地址设置是否正确
3. 确认防火墙没有阻止连接
4. 尝试在浏览器中直接访问后端地址

### 数据提取失败

**问题**：点击"提取数据"后没有反应或报错

**解决方案**：
1. 确认页面已完全加载
2. 检查是否在正确的页面（内容管理、视频详情等）
3. 查看浏览器控制台错误信息
4. 尝试刷新页面后重新提取

### 导入失败

**问题**：数据提取成功但导入失败

**解决方案**：
1. 检查后端日志查看具体错误
2. 确认后端同步接口正常工作
3. 检查数据格式是否正确
4. 确认数据库连接正常

## 🔒 隐私和安全

### 数据处理

- **本地处理**：所有数据提取都在浏览器本地完成
- **安全传输**：使用HTTPS加密传输到后端
- **数据存储**：扩展不会存储您的敏感数据

### 权限说明

扩展需要以下权限：

- `activeTab`：访问当前标签页内容
- `storage`：保存用户设置
- `scripting`：注入脚本到页面
- `host_permissions`：访问抖音创作中心和Agent后端

通过本扩展，您可以形成完整的数据闭环：

```
抖音创作中心 → 浏览器扩展 → Agent系统 → 数据分析 → 优化建议 → 抖音发布
```

## 🆘 技术支持

### 常见问题

**Q: 扩展支持哪些浏览器？**
A: 目前支持Chrome、Edge等基于Chromium的浏览器。

**Q: 可以提取历史数据吗？**
A: 可以，只要在抖音创作中心的历史页面，扩展就能提取数据。

**Q: 数据会重复导入吗？**
A: 不会，后端会根据内容去重，避免重复数据。

**Q: 扩展会收集我的个人信息吗？**
A: 不会，扩展只提取公开发布的数据，不会收集个人信息。
